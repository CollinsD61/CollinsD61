## 💫 About Me
I am an engineer passionate about building, optimizing, and deploying **Machine Learning (ML)**, **Deep Learning (DL)**, and **Large Language Model (LLM)** applications in Cloud environments. With a **DevOps / MLOps** mindset, I focus on automating the entire model lifecycle from training, optimization, to scalable production deployment.

- 🔭 **Current Focus:** Optimizing MLOps pipelines for GenAI, RAG, and LLM applications.
- 💡 **Expertise:** Designing CI/CD/GitOps architectures, infrastructure as code (IaC), and full-stack observability for AI systems.
- 🎯 **Goal:** Building highly scalable, low-latency, observable, and cost-efficient distributed AI systems on AWS.
- ⚡ **Fun fact:** *Training a massive model is like raising a kid—it takes tons of patience, continuous monitoring, and... a massive AWS budget.*
